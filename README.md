# Обзор Bash Скриптов

Этот проект содержит набор скриптов, выполненных на Bash, для решения различных задач, включая приветствие пользователя, генерацию пароля, работу с числами, поиск строк, создание структуры проекта и калькулятор.

---

## 1. Скрипт приветствия пользователя (`greeting.sh`)

**Описание:**  
Запрашивает у пользователя его имя и выводит персональное приветствие.

**Команда для запуска:**  
```bash
./greeting.sh
./password_generator.sh
./even_odd.sh
./create_project_structure.sh
./calculator.sh
./line_counter.sh
